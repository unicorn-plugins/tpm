# TPM 프로젝트 설정

## tpm 플러그인 라우팅 힌트

이 프로젝트는 tpm(Theater Production Manager) 플러그인을 사용합니다.

### 주요 명령
- `/tpm:propose` — 공연 기획 제안서 자동 생성 (전 과정 오케스트레이션)
- `/tpm:help` — 전체 명령 안내
- `/tpm:setup` — 초기 설정

### 자동 라우팅 키워드
다음 키워드가 포함된 요청은 tpm 플러그인으로 자동 라우팅됩니다:
- 공연 기획, 뮤지컬, 연극, 공연 제안서
- 예산 계획, 마케팅 전략, 시장 분석
- 포스터 이미지, 컨셉 이미지 생성
