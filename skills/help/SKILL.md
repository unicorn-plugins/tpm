# Help

[HELP 활성화]

## 목표

tpm 플러그인의 사용 가능한 명령과 자동 라우팅 키워드를 즉시 안내함.
추가 파일 탐색이나 에이전트 위임 없이, 아래 내용을 즉시 출력함.

## 활성화 조건

사용자가 `/tpm:help` 호출 시 또는 "도움말", "뭘 할 수 있어", "어떻게 써" 키워드 감지 시.

## 워크플로우

**중요: 추가적인 파일 탐색이나 에이전트 위임 없이, 아래 내용을 즉시 사용자에게 출력하세요.**

---

## TPM — 공연 기획 AI 팀

> 연극/뮤지컬 리서치부터 컨셉 이미지가 포함된 기획 제안서/PPT까지 전 과정을 자동화합니다.

### 슬래시 명령

| 명령 | 설명 |
|------|------|
| `/tpm:setup` | 플러그인 초기 설정 (generate_image 도구, GEMINI_API_KEY 설정) |
| `/tpm:propose` | 공연 기획 제안서 자동 생성 (6단계 전 과정) |
| `/tpm:help` | 이 사용 안내 출력 |
| `/tpm:add-ext-skill` | 외부호출 스킬 추가 |
| `/tpm:remove-ext-skill` | 외부호출 스킬 제거 |

### 자동 라우팅

다음 키워드가 포함된 요청은 자동으로 대응 스킬로 라우팅됩니다:

| 키워드 | 실행 스킬 |
|--------|----------|
| "기획", "제안서", "공연 기획", "뮤지컬 기획" | `/tpm:propose` |
| "설치", "설정", "초기화" | `/tpm:setup` |
| "도움말", "도움", "어떻게 써" | `/tpm:help` |

### 사용 예시

```
/tpm:propose
→ "어떤 공연을 기획하고 싶으신가요?"
→ "뮤지컬 '레미제라블' 한국 창작 버전"
→ 6단계 자동 실행 (리서치 → 시장분석 → 예산 → 제안서 → 이미지 → PPT)
```

### 팀 에이전트

| 에이전트 | 닉네임 | 역할 |
|----------|--------|------|
| researcher | 리서치 | 국내외 공연 자료 수집 및 레퍼런스 리서치 |
| market-analyst | 애널리스트 | 시장 분석 및 마케팅 전략 수립 |
| budget-planner | 예산관 | 항목별 예산 계획 및 BEP 분석 |
| proposal-writer | 라이터 | 기획 제안서 및 PPT 개요 작성 |
| visual-creator | 비주얼 | Gemini 기반 컨셉 이미지 생성 |

## MUST 규칙

| # | 규칙 |
|---|------|
| 1 | 파일 탐색이나 에이전트 위임 없이 즉시 출력 |
| 2 | 슬래시 명령 목록에 add-ext-skill, remove-ext-skill 포함 |

## MUST NOT 규칙

|---|----------|
