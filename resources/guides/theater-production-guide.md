# 공연 기획 도메인 전문 가이드

> tpm 플러그인 에이전트가 도메인 판단 시 참조하는 공연 기획 전문 지식 문서.

---

## 1. 공연 유형 분류

### 연극 (스트레이트 플레이)

**소극장 연극**
- 객석 규모: 100석 미만
- 특징: 배우와 관객의 친밀감, 실험적 작품 가능, 제작비 상대적으로 낮음
- 대표 공간: 대학로 소극장, 아트원씨어터 1관 등

**대극장 연극**
- 객석 규모: 500석 이상
- 특징: 웅장한 무대 연출, 앙상블 규모, 높은 제작비와 마케팅 비용 필요
- 대표 공간: 국립극장 해오름극장, LG아트센터, 세종문화회관 대극장 등

**스트레이트 연극**
- 음악 없이 대사 중심으로 진행되는 전통적 형식의 연극
- 원작 텍스트(희곡) 충실도가 중요

### 뮤지컬

**창작 뮤지컬**
- 국내 창작진이 대본·음악·가사를 새로 만든 작품
- 원작료 없음, 국내 정서에 맞는 소재 가능, IP 소유 가능
- 리스크: 검증되지 않은 작품성, 마케팅 난이도 높음

**라이선스 뮤지컬**
- 해외 원작의 공식 라이선스를 구매하여 제작
- 검증된 작품성, 해외 팬덤 활용 가능
- 리스크: 라이선스 비용, 원작사 간섭, 국내화 제약

**오리지널 뮤지컬 (내한 공연)**
- 해외 제작사의 오리지널 캐스트·스태프가 내한하여 공연
- 높은 화제성, 프리미엄 가격 책정 가능
- 리스크: 매우 높은 대관료 및 제작비, 짧은 공연 기간

### 음악극
- 클래식 음악 또는 민요를 활용한 연극적 형식
- 오페라, 가무극, 창극 등 포함

### 연극제 / 페스티벌
- 복수의 작품을 묶어 일정 기간 집중 선보이는 형식
- 서울연극제, 서울국제공연예술제(SPAF), 거리극 페스티벌 등
- 브랜드 구축 및 신진 작가 발굴 목적

## 2. 기획 프로세스 표준 절차

### 1단계: 작품 선정
- 기획 의도 및 시장성 검토
- 장르, 규모, 시기 결정
- 원작 유무 및 IP 현황 파악

### 2단계: 권리 확보
- 원작(소설, 영화, 웹툰 등) 공연화 권리 협상 및 계약
- 라이선스 뮤지컬의 경우 원작사와 라이선스 계약
- 저작권 범위, 기간, 지역 명확히 확인

### 3단계: 시장조사 및 타당성 검토
- KOPIS 데이터 기반 유사 공연 관객 수 분석
- 경쟁작 일정 및 시장 포화도 검토
- 타겟 관객 설문 또는 FGI(표적 집단 인터뷰)

### 4단계: 제작진 구성
- 연출가 섭외 및 계약
- 음악감독(뮤지컬), 안무가, 무대/조명/음향 디자이너 선정
- 주요 배우 캐스팅 협의

### 5단계: 예산 수립
- 항목별 상세 예산 책정
- 투자 유치 계획 또는 자체 제작 여부 결정
- 손익분기점(BEP) 분석

### 6단계: 마케팅 계획 수립
- 마케팅 콘셉트 및 슬로건 개발
- 채널별 광고 계획 및 예산 배분
- 티켓 가격 정책 결정 (등급별, 얼리버드, 기획 할인 등)

### 7단계: 프리프로덕션
- 극장 대관 계약 확정
- 대본 완성 및 배역 확정
- 무대·의상·소품 제작 착수
- 연습실 섭외 및 연습 일정 수립

### 8단계: 공연 (Production)
- 기술 리허설(테크니컬 리허설) → 드레스 리허설 → 프레스콜 → 공식 개막
- 공연 기간 중 마케팅 지속 운영
- 관객 반응 모니터링 및 즉시 대응

### 9단계: 정산 및 평가
- 공연 종료 후 수익 정산 및 투자자 배분
- 성과 리포트 작성 (관객 수, 매출, 언론 반응)
- 후속 기획(재공연, 투어, 시리즈화) 검토

## 6. 관객 규모 추정 공식

### 기본 공식

```
최대 관객 수 = 좌석 수 × 총 회차 수
예상 관객 수 = 최대 관객 수 × 목표 객석 점유율
```

### 시나리오별 추정

| 시나리오 | 점유율 기준 | 설명 |
|---------|-----------|------|
| 보수 | 50~60% | 비수기, 검증되지 않은 신작, 마케팅 미흡 시 |
| 기본 | 70~80% | 일반적 흥행 목표 |
| 낙관 | 90% 이상 | 스타 캐스팅, 화제작, 성수기 집중 시 |

### 손익분기점(BEP) 계산

```
BEP 관객 수 = 총 고정비 ÷ 1인당 평균 공헌이익
1인당 평균 공헌이익 = 평균 티켓 가격 - 변동비 (1인당)
BEP 점유율 = BEP 관객 수 ÷ 최대 관객 수 × 100%
```

### 매출 추정

```
예상 매출 = 예상 관객 수 × 평균 티켓 가격 × (1 - 예매 수수료율)
예매 수수료율: 인터파크/YES24 약 8~10%
```

## 7. 공연 기획 용어 사전

| 용어 | 설명 |
|------|------|
| **프리프로덕션 (Pre-production)** | 공연 전 모든 제작 준비 과정. 대본 완성, 제작진 계약, 무대·의상 제작, 연습실 연습 등 |
| **테크니컬 리허설 (Tech Rehearsal)** | 무대, 조명, 음향, 영상 등 기술 요소를 통합하여 극장에서 진행하는 기술 리허설 |
| **드레스 리허설 (Dress Rehearsal)** | 의상·분장·소품까지 완비하고 실제 공연과 동일한 조건으로 진행하는 최종 리허설 |
| **프레스콜 (Press Call)** | 언론 및 미디어를 대상으로 공개하는 시연 행사. 포토콜과 하이라이트 장면 시연 포함 |
| **투자자 쇼케이스** | 잠재적 투자자를 대상으로 작품 일부를 선보이고 투자를 유치하는 행사 |
| **티켓 파워 (Ticket Power)** | 특정 배우 또는 작품이 티켓 판매에 미치는 영향력. 팬덤이 강한 배우의 티켓 파워가 높음 |
| **대관료** | 극장을 일정 기간 빌리는 비용. 고정 대관료 방식과 매출 연동 방식이 있음 |
| **그랜드 오픈 (Grand Open)** | 공식 개막 공연. 초연(初演) 또는 오프닝 나잇(Opening Night)이라고도 함 |
| **KOPIS** | 공연예술통합전산망(Korea Performing Arts Box Office Information System). 공연 통계 공식 집계 기관 |
| **BEP (Break Even Point)** | 손익분기점. 총 수입이 총 비용과 같아지는 지점. 이 이상 관객이 와야 흑자 |
| **러닝 개런티 (Running Guarantee)** | 공연 회차 또는 관객 수에 따라 배우·스태프에게 추가 지급하는 성과 보수 |
| **앙상블 (Ensemble)** | 뮤지컬에서 주요 배역 외 여러 역할을 소화하는 배우군. 무용·노래·연기를 두루 수행 |
| **스윙 (Swing)** | 특정 배역에 고정되지 않고 여러 앙상블 역할을 대체할 수 있는 배우 |
| **오픈런 (Open Run)** | 종료일을 미리 정하지 않고 관객 수요에 따라 공연을 이어가는 방식 |
| **내한 공연** | 해외 제작사의 오리지널 프로덕션이 국내에 들어와 공연하는 형태 |
| **창작 뮤지컬** | 국내 창작진이 대본과 음악을 새로 만든 뮤지컬 |
| **라이선스 뮤지컬** | 해외 원작의 공식 공연권을 취득하여 국내에서 제작한 뮤지컬 |
| **포토콜 (Photo Call)** | 언론사 및 포토그래퍼를 대상으로 공식 촬영 기회를 제공하는 행사 |
| **굿즈 (Goods)** | 공연 관련 기념 상품. 포토북, 브로마이드, 에코백, 키링 등 |
| **인터미션 (Intermission)** | 막간 휴식 시간. 대형 뮤지컬의 경우 보통 15~20분 |
