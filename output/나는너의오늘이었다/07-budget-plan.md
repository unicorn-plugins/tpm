## 4. 손익분기점 분석

### 기본 전제

| 항목 | 수치 |
|------|------|
| 총 좌석 수 | 150석 |
| 총 공연 회차 | 48회 |
| 최대 관객 수 | 7,200명 (150 × 48) |
| 티켓 평균 단가 | 50,000원 |
| 예매 수수료율 | 9% |
| 1인당 순 티켓 수입 | 45,500원 (50,000원 × (1 - 9%)) |
| 변동비 (1인당) | 500원 (향기 키트 소모품 단가. 회당 배포 원가 개념) |
| **1인당 평균 공헌이익** | **45,000원** |

### BEP 계산

```
BEP 관객 수 = 총 고정비 ÷ 1인당 평균 공헌이익

총 고정비 = 총 예산 - 향기 키트 변동분
         = 19,000,000원 - (7,200명 × 500원 최대 변동비)
         = 19,000,000원 - 3,600,000원 (최대 변동 추정)

※ 보수적 BEP 산정을 위해 총 예산 전액(19,000,000원)을 고정비로 취급

BEP 관객 수 = 19,000,000원 ÷ 45,000원 = 422.2명 → 약 423명
BEP 점유율 = 423명 ÷ 7,200명 × 100% ≒ 5.9%
```

| 지표 | 수치 |
|------|------|
| 총 고정비 (보수적) | 19,000,000원 |
| 1인당 평균 공헌이익 | 45,000원 |
| **BEP 관객 수** | **423명** |
| **BEP 점유율** | **5.9%** |
| 1회 평균 BEP 관객 | 8.8명/회 |

> **[BEP 해석]** 총 7,200석 중 423명(전체의 5.9%)만 유료 입장해도 원가 회수가 가능한 구조입니다.
> 극단히 낮은 BEP는 소극장 규모와 합리적 예산 통제의 결과입니다.
> 단, 실제로는 '수익 확보'를 위해 최소 50% 이상 점유율(3,600명 이상)을 목표로 운영해야 합니다.

| 퍼펫/특수 의상 | 500,000원 | +300,000원 | 일반 의상비 대비 추가 |
| **특수 연출 추가 합계** | **2,000,000원** | **+1,500,000원** | |

> **[조정 결과]** 특수 연출 비용 추가로 인해 공연 기간을 8주(64회)에서 6주(48회)로 단축 조정.
> 이를 통해 총 예산을 2,000만 원 이내로 유지하면서 작품의 핵심 차별화 요소를 모두 반영함.
