### 4-2. 공헌이익 계산

```
회당 공헌이익 = 공연 단가 - 회당 변동비
             = 900,000원 - 435,000원
             = 465,000원/회
```

### 4-3. 손익분기점 (BEP) 분석

```
BEP 회차 = 초기 고정 투자비 ÷ 회당 공헌이익
         = 8,955,000원 ÷ 465,000원
         = 19.3회 → 약 20회
```
