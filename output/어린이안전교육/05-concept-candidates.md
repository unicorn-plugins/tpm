## 수렴된 공연 컨셉 후보 목록 (총 4개)

### 후보 1: 「위험을 막아라, 탐정단!」 — 선택형 추리 뮤지컬 + 인식 변화 즉석 리포트

- **핵심 경험 가치**: 어린이가 극 중 위기 상황에서 투표·선택으로 주인공(홍길동 탐정)의 행동을 결정하고, 올바른 안전 행동이 무대 위에서 즉각 실현·검증되는 경험. 공연 전후 OX 퀴즈(10문항)로 학급의 안전 인식 점수 변화를 즉석 리포트로 교사에게 제공
- **공연 형태**: 분기형 인터랙티브 뮤지컬 + 포럼 씨어터 혼합 / 배우 3인(홍길동·김영희·조연) / 50분 공연 + 전후 15분 평가 / 이동형 학교 강당
- **차별화 포인트**: 분기형 서사+포럼 씨어터 결합(국내 최초), 공연 직후 학급별 인식 변화 점수 리포트 제공, 연 2회 계약 유인(회차마다 다른 결말)
- **Needs Statement 연결**: 사전·사후 OX 리포트가 변화를 수치로 시각화 → 교사의 재계약·상급자 보고 근거 직결
- **타 분야 벤치마킹**: Nintendo 뇌 나이 진단(사전·사후 측정 루프), Big Brum TIE 포럼 씨어터(관객 발언 배우 재연), 키자니아(배지·인증서 보상)
- **출처 아이디어**: MA-Big1 + PW-Big1 + MA-Big3 합침
- **전속 배우 배역**: 홍길동(수석 탐정 역), 김영희(탐정 파트너 겸 포럼 씨어터 재연 역)
- **예상 제작비**: 1,500~2,000만 원 / 제작 기간 8~10주

### 후보 2: 「빛나는 대피로」 — 프로젝션 매핑 체험극

- **핵심 경험 가치**: 어린이가 올바른 대피 행동을 선택할 때 교실·강당 벽면 프로젝션이 실시간으로 안전한 공간으로 변화하는 시각적·신체적 몰입 경험
- **공연 형태**: 프로젝션 매핑 인터랙티브 체험극 / 배우 2인(홍길동·김영희) + 영상 오퍼레이터 / 40분 / 이동형 (소형 프로젝터 2대)
- **차별화 포인트**: 기술 기반 감각적 시각화(행동 선택 결과가 공간 전체에 반응), 신체 이동 대피 체험(앉아서 보는 관람 탈피), 교사 관찰 가능성(어린이 행동 외침·이동 속도)
- **Needs Statement 연결**: 선택의 결과가 공간 시각화로 즉각 드러남. 전후 구술 카드로 지식 변화 확인
- **타 분야 벤치마킹**: teamLab Borderless(관객 행동이 환경 실시간 변화), 일본 극단 비행선 재난 대피 체험, 소방청 AR 대피 체험 훈련
- **출처 아이디어**: MA-Big2 독립 유지
- **전속 배우 배역**: 홍길동(어린이 선택을 따라 행동하는 길잡이), 김영희(안전 행동 이유를 설명하는 해설자)
- **예상 제작비**: 2,000~2,500만 원 / 제작 기간 10~12주

### 후보 3: 「위험한 어른과 똑똑한 아이들」 — 역할 역전 참여극

- **핵심 경험 가치**: 홍길동이 "위험한 행동을 반복하는 어른"으로 등장하고, 어린이 관객 전원이 "안전 교관"으로 홍길동을 교정한다. 아이들이 안전 지식을 소리 내어 말하는 순간 자체가 교육 효과의 직접 증거
- **공연 형태**: 역할 역전 포럼 씨어터 / 배우 2인(홍길동·김영희) / 45분 / 이동형 학교 강당
- **차별화 포인트**: 파이넌만 기법(가르침=최고의 학습), 배우 2인 최소 구성으로 제작비·단가 현실화, 홍길동의 코미디 연기로 웃음+안전 각인 동시 달성
- **Needs Statement 연결**: 어린이의 자발적 발화가 인식의 직접 표출. 관찰 체크리스트로 장면별 응답 수준 즉시 확인
- **타 분야 벤치마킹**: Big Brum TIE 포럼 씨어터, 미국 Chicago Children's Theatre TIE, 파이넌만 기법(가르침을 통한 학습 내재화)
- **출처 아이디어**: PW-Big2 + MA-Crazy 법정극의 "어린이 판결 선언" 구조 부분 흡수
- **전속 배우 배역**: 홍길동(위험한 행동을 반복하는 코미디 어른), 김영희(걱정하는 친구 겸 진행자)
- **예상 제작비**: 1,000~1,400만 원 / 제작 기간 6~8주

### 후보 4: 「안전 여행자의 비밀 지도」 — 3막 옴니버스 뮤지컬 + 안전 여권

- **핵심 경험 가치**: 어린이 한 명 한 명이 안전 여권 소책자를 손에 쥐고, 3개 나라(화재·교통·지진)를 여행하며 미션 클리어 시 스탬프를 받는다. 여권이 가정 연계 교육 매개체로 확장
- **공연 형태**: 3막 옴니버스 뮤지컬 / 배우 3인(홍길동·김영희·조연) / 50분 공연 + 스탬프 세리머니 / 이동형 학교 강당
- **차별화 포인트**: 유형적 학습 결과물(여권 소책자), QR 코드 후속 콘텐츠(가정 미션 완료 시 배우 영상), 뮤지컬 넘버 3곡으로 음악적 기억 각인
- **Needs Statement 연결**: 여권 스탬프 완성이 학습 여정 물리적 시각화. QR 코드 접속 수로 사후 데이터 제공
- **타 분야 벤치마킹**: 키자니아 스탬프·인증서(반복 방문 유인), 듀오링고 스트릭·가정 연계 미션(행동 지속 유도), 박물관 스탬프 투어(공연 적용)
- **출처 아이디어**: PW-Big3 + MA-Little Win2 뮤지컬 넘버·QR 코드 요소 보강
- **전속 배우 배역**: 홍길동(성장하는 여행자 주인공), 김영희(나라별 안전 안내인 — 막마다 다른 의상 변신)
- **예상 제작비**: 1,800~2,200만 원 / 제작 기간 10~12주
