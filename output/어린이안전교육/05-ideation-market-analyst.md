## 2. SCAMPER 적용 현황

| 기법 | 적용 내용 | 해당 아이디어 |
|------|----------|------------|
| **S - Substitute** | 기존 "배우의 일방적 연기"를 "어린이가 직접 조종하는 선택지 극"으로 대체 | Big Idea 1, Crazy Idea |
| **C - Combine** | 뮤지컬 + 진단 평가 도구를 결합하여 "공연 자체가 측정 도구"인 형식 창출 | Big Idea 2 |
| **A - Adapt** | 키자니아의 역할 체험 + 스탬프 수집 구조를 학교 방문형 공연에 적용 | Big Idea 3 |
| **M - Modify** | 60분 단일 공연을 "10분 x 4 에피소드" 모듈형으로 시간 구조 변형 | Little Win Idea 1 |
| **E - Eliminate** | 고정 무대/조명/음향 장비를 제거하고 배우 신체 + 소품 가방 1개로 극한 경량화 | Little Win Idea 2 |
| **R - Reverse** | 관객(어린이)이 배우에게 안전 지식을 "가르치는" 역발상 구조 | Crazy Idea |
| **P - Put to other uses** | 학교 수업용 교육 공연을 가족 체험형 공연(B2C) + 교사 연수(B2B2) 3중 활용 | Big Idea 3 |

## 3. 아이디어 목록

### Big Idea 1: "세이프 레벨업 -- 선택이 바꾸는 안전 세계"

| 항목 | 내용 |
|------|------|
| **구분** | Big Idea |
| **컨셉 제목** | 세이프 레벨업 -- 선택이 바꾸는 안전 세계 |
| **컨셉 설명** | 어린이 관객이 극 중 위기 상황에서 다수결 투표(손 들기 or 카드 선택)로 주인공의 행동을 결정하는 **분기형 인터랙티브 뮤지컬**. 화재/교통/지진/학교폭력 4개 위기 장면이 등장하며, 관객의 선택에 따라 결말이 달라진다. 공연 전 "안전 레벨 테스트(10문항 OX)"를 실시하고, 공연 후 동일 테스트를 재실시하여 학급 전체의 안전 인식 점수 변화를 교사에게 즉석 리포트로 제공한다. |
| **SCAMPER/Steal 연결** | S(Substitute): 수동 관람 -> 관객 선택형 극 / S1(게임: 뇌 나이 진단-피드백 루프) / S4(teamLab: 관객 행동이 환경 변화) |
| **핵심 경험 가치** | 교사가 공연 직후 "우리 반 안전 인식이 몇 점 올랐다"는 수치를 확인할 수 있어, Needs Statement의 "눈에 보이는 변화"를 가장 직접적으로 충족. 어린이는 자기 선택이 결과에 영향을 미치는 경험을 통해 안전 행동의 주체성을 체감. |
| **공연 형태** | 참여형 뮤지컬 / 배우 4인 + 진행자(MC) 1인 / 50분 공연 + 15분 사전사후 평가 / 이동형 학교 방문 |
| **차별화 포인트** | (1) 국내 최초 "교육 효과 즉석 측정 내장형" 공연 -- 교사에게 정량 데이터 제공 (2) 4개 안전 주제 통합 -- 1회 방문으로 복합 안전교육 의무 시수 충족 (3) 분기형 서사 -- 반복 관람 시 다른 결말 경험 가능(연 2회 계약 유인) |
| **Needs Statement 연결** | "안전 인식 변화가 눈에 보이는" -> 사전사후 안전 레벨 점수 비교 리포트가 핵심 전달물. "신뢰할 수 있는 의사결정" -> 정량 데이터가 재계약/추천 근거 제공. |
| **시장 매력도** | B2B 계약률 극대화 요인: 교육 효과 증빙 보고서 자동 생성으로 교사의 행정 부담 해소. 교육청 공모사업 선정률 우위 예상(효과 측정 체계 내장). 예상 회당 계약가 150~200만 원(무료 공공 공연 대비 프리미엄 정당화 가능). |

### Big Idea 2: "안전 탐정단 -- 사건을 해결하라"

| 항목 | 내용 |
|------|------|
| **구분** | Big Idea |
| **컨셉 제목** | 안전 탐정단 -- 사건을 해결하라 |
| **컨셉 설명** | 학교 강당이 하나의 "사건 현장"으로 변환되는 **포럼 씨어터 기반 추리 참여극**. 주인공 '안전 탐정'이 학교에서 벌어진 연쇄 안전사고(화재 오보, 횡단보도 사고, 지진 대피 혼란, 친구 간 갈등)의 원인을 추적하며, 어린이 관객이 "탐정단원"으로 참여하여 단서를 모으고 올바른 안전 행동을 제안한다. 공연 중 어린이가 제안한 안전 행동은 배우가 즉석에서 재연하여 "이렇게 하면 사고가 예방되었을 것"을 시각화한다. 공연 후 각 반에 "탐정 보고서"(학습 워크시트)를 배부하여 교과 시간에 연계 수업이 가능하도록 설계. |
| **SCAMPER/Steal 연결** | C(Combine): 추리극 + 포럼 씨어터 + 안전교육 3중 결합 / Big Brum(영국) 포럼 씨어터 방법론 적용 / S2(키자니아: 역할 수행 + 보상) |
| **핵심 경험 가치** | 어린이가 "안전 문제를 발견하고 해결책을 스스로 제안하는" 능동적 경험. 교사는 어린이들의 발언/제안 내용을 통해 안전 인식 수준을 실시간 관찰 가능. 추리극 포맷이 주는 몰입감으로 교통/화재/지진/학교폭력이라는 교훈적 소재의 "설교적 느낌"을 해소. |
| **공연 형태** | 포럼 씨어터 기반 참여형 연극 / 배우 3인 + 진행자 1인 / 60분(공연 45분 + 탐정 보고서 작성 15분) / 이동형 학교 방문 |
| **차별화 포인트** | (1) 국내 유일의 "추리극 형식 안전교육" -- 재미와 교육의 균형 극대화 (2) 포럼 씨어터 방식으로 어린이 발언을 배우가 즉석 재연 -- 영국 Big Brum 검증 방법론의 국내 최초 안전교육 적용 (3) "탐정 보고서" 교과 연계 워크시트로 일회성 탈피 |
| **Needs Statement 연결** | "안전 인식 변화가 눈에 보이는" -> 어린이가 직접 제안한 안전 행동이 무대 위에서 시각화됨(변화의 가시화). "실질적 효과 확인" -> 탐정 보고서의 답변 내용이 교육 효과의 정성적 증거. |
| **시장 매력도** | 교사 네트워크 입소문 효과 극대화 예상: "아이들이 직접 발언하는 공연"이라는 체험 차별성이 교사 간 구전 유도. 교육청 공모사업 "참여형 문화예술교육" 카테고리 적합. 추리극 형식은 초등 3~4학년까지 타겟 확장 가능(부타겟 확대). |

### Big Idea 3: "안전 히어로 스쿨 -- 4주 여정"

| 항목 | 내용 |
|------|------|
| **구분** | Big Idea |
| **컨셉 제목** | 안전 히어로 스쿨 -- 4주 여정 |
| **컨셉 설명** | 1회성 공연이 아닌 **"4주 패키지 프로그램"**으로 설계된 시즌제 안전교육 뮤지컬. [1주차] 사전 학습 워크시트 배부 + "안전 히어로 카드" 배급 / [2주차] 본 공연(뮤지컬 50분, 안전 영역별 에피소드 4개) / [3주차] 가정 연계 미션("이번 주 안전 미션: 가족과 화재 대피 경로 확인하기") / [4주차] 배우 영상 메시지 + 최종 "안전 히어로 인증서" 발급 및 사전사후 비교 리포트 제공. 4주간의 변화 추적이 가능하여 교사에게 가장 강력한 교육 효과 증빙 자료를 제공. |
| **SCAMPER/Steal 연결** | A(Adapt): 키자니아 스탬프 수집 + 듀오링고 스트릭 구조를 공연에 적용 / P(Put to other uses): 학교 수업 / 가정 교육 / 교사 보고서 3중 용도 / S2(키자니아 인증서) / S3(듀오링고 스트릭 & 트래킹) |
| **핵심 경험 가치** | 교사는 4주간의 데이터(사전 퀴즈 -> 공연 후 퀴즈 -> 가정 미션 완수율 -> 최종 평가)를 통해 "교육 효과의 시간적 변화"를 추적 가능. Needs Statement의 "실질적 효과 확인"을 가장 강력하게 충족. 어린이는 4주간의 "성장 여정"을 경험하며, 단순 관람이 아닌 "나의 안전 실력이 자란다"는 성취감을 얻음. |
| **공연 형태** | 시즌제 패키지 프로그램 (공연 1회 + 사전/사후 학습자료 + 가정 미션 + 인증서) / 배우 4인 / 본 공연 50분 / 이동형 학교 방문 |
| **차별화 포인트** | (1) 국내 안전교육 공연 시장 최초의 "패키지 프로그램" -- 1회성 공연 관행 파괴 (2) 4주간 교육 효과 트래킹 데이터 제공 -- 교육청 보고서 작성에 바로 활용 가능 (3) 가정 연계 미션으로 학부모 참여 유도 -- B2C 전환(주말 가족 공연) 자연스러운 접점 생성 |
| **Needs Statement 연결** | "눈에 보이는 교육 경험" -> 4주간의 변화 데이터가 시간축 위에 시각화됨. "신뢰할 수 있는 의사결정" -> 4주 데이터는 1회 사전사후 측정보다 교육학적 신뢰도가 높아 재계약 근거로 강력. |
| **시장 매력도** | 회당 단가가 아닌 "패키지 단가" 설정 가능(250~400만 원 예상) -- 매출 단가 상승. 재계약률 극대화 요인: 4주 데이터를 경험한 교사는 다음 해에도 동일 프로그램 발주 가능성 높음. 교육청 "문화예술교육 + 안전교육" 융합 공모사업에 최적합 포맷. 제작비는 본 공연 1회분이므로 추가 자료 제작비만 증분(총 제작비 2,000만 원 이내 가능). |

### Little Win Idea 1: "안전 뉴스 -- 오늘의 긴급 속보!"

| 항목 | 내용 |
|------|------|
| **구분** | Little Win Idea |
| **컨셉 제목** | 안전 뉴스 -- 오늘의 긴급 속보! |
| **컨셉 설명** | 학교 강당을 "뉴스 스튜디오"로 꾸미고, 배우 2인이 앵커와 현장 기자 역할을 맡아 4개 안전 주제(교통/화재/지진/학교폭력)를 뉴스 리포트 형식으로 전달하는 **모듈형 교육극**. 각 "뉴스 꼭지"는 10분 단위로 독립 구성되어, 학교 일정에 따라 2개만 선택 공연(20분)하거나 4개 모두 공연(40분)하는 등 시간 유연성을 제공. 각 꼭지 끝에 "긴급 퀴즈 속보"로 어린이 참여 유도. 공연 후 "오늘의 안전 뉴스 성적표"를 학급에 전달. |
| **SCAMPER/Steal 연결** | M(Modify): 60분 단일 서사 -> 10분 x 4 모듈로 시간 구조 변형 / E(Eliminate): 복잡한 서사/캐릭터 설정 제거, 뉴스 포맷의 간결한 구조 |
| **핵심 경험 가치** | 교사가 수업 시간에 맞춰 공연 길이를 조절할 수 있어 학교 시간표 갈등 해소(현장 도입 장벽 최소화). 뉴스 형식이 어린이에게 친숙하면서도 정보 전달 효율이 높음. 꼭지별 퀴즈 결과가 간이 성적표로 교사에게 전달. |
| **공연 형태** | 모듈형 교육극 / 배우 2인 / 20~40분(모듈 선택) / 이동형 학교 방문 / 간이 뉴스 데스크 세트(접이식) |
| **차별화 포인트** | (1) 모듈형 -- 학교 일정에 맞춘 유연한 공연 시간 (2) 배우 2인 최소 구성 -- 회당 비용 80~120만 원대 가능(가격 경쟁력) (3) 뉴스 포맷 -- 기존 뮤지컬/연극과 완전히 다른 형식적 신선함 |
| **Needs Statement 연결** | "눈에 보이는 변화" -> 꼭지별 퀴즈 성적표로 간이 효과 측정. "실질적 효과" -> 모듈형이라 학교 교과 시간에 바로 삽입 가능하여 실제 수업 연계성 높음. |
| **시장 매력도** | 빠른 시장 진입 가능: 제작 기간 4~6주, 제작비 800~1,200만 원 예상. 배우 2인 구성으로 1일 2~3개교 순회 가능(운영 효율 극대화). 모듈형이라 특정 안전 주제만 원하는 학교도 계약 가능(시장 유연성). 무료 공공 공연 대비 "시간 유연성"이라는 비가격 차별화로 유료 정당화. |

### Little Win Idea 2: "안전 가방극 -- 소품 가방 하나로 떠나는 안전 여행"

| 항목 | 내용 |
|------|------|
| **구분** | Little Win Idea |
| **컨셉 제목** | 안전 가방극 -- 소품 가방 하나로 떠나는 안전 여행 |
| **컨셉 설명** | 배우 1인이 커다란 빨간 가방 하나를 들고 교실에 등장하여, 가방에서 소품(소화기 모형, 미니 신호등, 지진 헬멧, 우정 팔찌 등)을 하나씩 꺼내며 안전 이야기를 풀어가는 **1인극(원맨쇼) + 체험 워크숍** 형식. 소품을 어린이에게 직접 건네 역할극에 참여시키고, 각 소품 체험 후 "안전 스티커"를 아이들의 워크시트에 붙여준다. 4개 소품 = 4개 안전 주제. 워크시트에 모든 스티커가 채워지면 "안전 여행 완료 도장"을 찍어준다. |
| **SCAMPER/Steal 연결** | E(Eliminate): 무대/조명/음향/복수 배우 모두 제거. 교실 단위 소규모 공연으로 극한 경량화 / S2(키자니아: 스탬프 수집 + 인증) |
| **핵심 경험 가치** | 강당이 아닌 "교실"에서 진행 가능 -- 강당 일정 확보 부담 없이 학급 단위로 운영. 어린이가 소품을 직접 만지고 체험하는 촉각적 경험으로 기억 잔존율 극대화. 교사는 스티커 워크시트의 완성 여부로 간편하게 참여도/이해도 파악. |
| **공연 형태** | 1인극 + 체험 워크숍 / 배우 1인 / 30분(공연 20분 + 체험 10분) / 학급 단위(교실 방문) / 소품 가방 1개로 이동 |
| **차별화 포인트** | (1) 배우 1인 + 가방 1개 -- 극한의 이동성과 비용 효율(회당 40~60만 원대 가능) (2) 교실 단위 운영 -- 강당 미확보 학교도 계약 가능(시장 확장) (3) 소품 체험 -- 유치원~초등 1학년 대상 최적화(이 연령대는 시각/촉각 결합 교육이 효과적) |
| **Needs Statement 연결** | "눈에 보이는 변화" -> 워크시트 스티커 완성 과정이 아이의 학습 여정을 시각적으로 보여줌. "신뢰할 수 있는 의사결정" -> 1학급 30분, 저비용이라 시범 도입 후 전 학년 확대 결정이 용이(리스크 낮은 의사결정). |
| **시장 매력도** | 시장 진입 속도 최고: 제작 기간 2~3주, 제작비 400~600만 원. 1일 6~8학급 순회 가능(1개 학교에서 하루에 전 학년 커버). 유치원/어린이집 시장까지 확장 가능(전국 8,500여 곳). "일단 저비용으로 시작해서 교사 반응 데이터를 축적"하는 MVP(최소 기능 제품) 전략에 최적합. 이 데이터를 기반으로 Big Idea로 스케일업하는 2단계 전략 가능. |

### Crazy Idea: "거꾸로 안전 학교 -- 아이들이 선생님을 가르치는 날"

| 항목 | 내용 |
|------|------|
| **구분** | Crazy Idea |
| **컨셉 제목** | 거꾸로 안전 학교 -- 아이들이 선생님을 가르치는 날 |
| **컨셉 설명** | 공연의 관객-배우 관계를 완전히 뒤집는 **역발상 참여극**. 배우들이 "안전에 대해 아무것도 모르는 외계인 전학생" 역할로 등장하고, 어린이 관객 전원이 "안전 선생님"이 되어 외계인에게 안전 규칙을 가르쳐야 한다. 외계인은 일부러 위험한 행동을 시도하고("불이 나면 엘리베이터를 타면 되지?"), 아이들이 "아니야!"라고 외치며 올바른 행동을 설명하면 외계인이 "배우며" 변화한다. 극의 종반부에는 외계인이 완벽한 안전 행동을 수행하며, 아이들에게 "선생님 덕분에 배웠어요"라고 감사 인사를 한다. **핵심 반전:** 아이들이 외계인을 "가르치는" 과정이 곧 아이들 자신의 학습이며, 교사는 아이들의 발언 내용을 통해 안전 인식 수준을 직접 관찰할 수 있다. 공연 후 "외계인이 보내온 편지"(감사 카드 + 안전 퀴즈)를 학급에 전달하여 후속 활동 연결. |
| **SCAMPER/Steal 연결** | R(Reverse): 배우=학생, 관객=선생님으로 역할 완전 역전 / S(Substitute): 기존 "가르치는 공연" -> "아이가 가르치는 공연"으로 대체 / S1(게임: 튜토리얼에서 플레이어가 NPC를 가르치는 구조) |
| **핵심 경험 가치** | "가르치는 것이 가장 좋은 학습"이라는 교육학 원리(Feynman Technique)를 공연에 구현. 아이들이 수동적 학습자가 아닌 능동적 교수자 역할을 경험하며 안전 지식의 내재화 극대화. 교사는 아이들의 "가르침" 내용을 관찰함으로써 사전 지식 수준과 오개념을 즉시 파악 가능 -- Needs Statement의 "안전 인식 변화가 눈에 보이는" 경험의 가장 파격적 구현. |
| **공연 형태** | 역발상 참여극 / 배우 2~3인(외계인 역) + 진행자 1인 / 50분 / 이동형 학교 방문 |
| **차별화 포인트** | (1) 국내외 안전교육 공연에서 전례 없는 "역할 역전" 구조 -- 차별화 임팩트 극대화 (2) 어린이의 자발적 발언이 공연의 핵심 콘텐츠 -- 매 회차가 유일무이한 공연 (3) "외계인" 캐릭터가 SNS 바이럴 잠재력 보유(B2C 전환 시 캐릭터 IP 활용 가능) |
| **Needs Statement 연결** | "안전 인식 변화가 눈에 보이는" -> 아이들이 외계인을 "가르치는" 발언 자체가 안전 인식의 가장 직접적인 표출. 공연 초반(외계인이 모르는 상태에서 아이들의 첫 반응)과 종반(외계인의 실수를 아이들이 즉각 교정하는 속도)의 차이가 교사에게 변화를 "눈으로" 보여줌. |
| **시장 매력도** | 파격적 컨셉으로 교육 매체/SNS 보도 가능성 높음 -- 무료 PR 효과. "외계인이 우리 학교에 왔다"는 스토리가 아이들의 학교생활 화제가 되어 학부모 인지도 자연 확산. 성공 시 "안전 외계인" 캐릭터 IP로 확장 가능(그림책, 유튜브, 굿즈). 리스크: 어린이 참여도가 낮을 경우 극이 진행되지 않을 수 있으므로 진행자의 퍼실리테이션 역량이 핵심 성공 요인. |

## 5. 전략적 제언

### 시장 진입 시퀀스 제안

**[Phase 1: MVP 검증 -- 2개월]**
Little Win Idea 2 "안전 가방극"으로 빠르게 시장 진입. 제작비 400~600만 원으로 10~20개교 시범 운영. 교사 반응 데이터 및 어린이 워크시트 결과 축적.

**[Phase 2: 주력 상품 런칭 -- 4개월]**
Phase 1 데이터를 기반으로 Big Idea 1 "세이프 레벨업" 또는 Big Idea 3 "안전 히어로 스쿨" 중 택 1 제작. 사전사후 측정 시스템 내장. 교육청 공모사업 제안서에 Phase 1 실증 데이터 포함.

**[Phase 3: 포트폴리오 확장 -- 6개월 이후]**
Crazy Idea "거꾸로 안전 학교"를 파일럿 운영하여 차별화 컨셉 테스트. 성공 시 캐릭터 IP 확장(B2C 가족 공연 전환).

### 킹핀 문제(P1) 해결력 순위

1. **Big Idea 3 "안전 히어로 스쿨"** -- 4주 데이터로 교육 효과 가시화 수준 최고
2. **Big Idea 1 "세이프 레벨업"** -- 즉석 정량 리포트로 즉시 가시화
3. **Crazy Idea "거꾸로 안전 학교"** -- 어린이 발언이 곧 인식의 직접 표출이라는 독창적 가시화

> **애널리스트 소견:** 6개 아이디어 모두 Needs Statement의 핵심인 "교육 효과의 가시화"를 각기 다른 방식으로 구현한다. 시장 관점에서 가장 중요한 판단 기준은 **"교사가 교육 효과를 확인하는 데 걸리는 시간"**이다. Big Idea 1은 공연 직후 65분 만에, Big Idea 3은 4주에 걸쳐, Crazy Idea는 공연 중 실시간으로 효과를 보여준다. 교사의 인내심과 행정 여건을 고려하면 "즉시 보여주는" Big Idea 1과 "저비용으로 빠르게 검증하는" Little Win Idea 2의 조합이 초기 시장 진입에 가장 유리하다고 판단한다.
